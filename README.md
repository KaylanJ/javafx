# Instructions for DatabaseManager JavaFX file

#### Before you run the JavaFX, you need a MariaDB database in the same structure outlined in the Database Setup file.

#### You should populate the tables before running the JavaFX.

#### Before executing the file, change the variables that are just "#####" to the proper string needed.

#### Otherwise you should be able to use the JavaFX, keeping in mind some functions are unfinished so you would need to populate manually
